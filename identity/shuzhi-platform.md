## 一、平台定位

**火山引擎数智平台（Volcano Engine Data Intelligence，简称 VeDI）** 是火山引擎推出的 **新一代企业数据智能平台**，基于字节跳动多年"数据驱动"实践经验，汇集端到端数智产品、场景化行业解决方案及专业数智转型咨询，帮助企业构建从数据基础建设、引擎升级到智能应用的全链路数智能力，实现 **运维降本、运营提效与业务增长**，助力数智化转型升级。

---

## 二、核心能力体系（四层架构）

### 1. 数据引擎层
提供云原生数据仓库 **ByteHouse**、流式计算 **Flink 版**、AI 数据湖服务 **LAS** 等基础数据处理能力，支撑大规模数据存储与计算。

### 2. 数据建设层
通过大数据研发治理套件 **DataLeap**、全域数据集成 **DataSail** 等产品，实现数据的高效采集、治理与集成。

### 3. 数据应用层
覆盖 **智能分析 Agent、智能营销 Agent** 等 AI 智能体服务，以及 **智能数据洞察 DataWind、管理驾驶舱 Plus** 等可视化分析工具。还包括传统数据套件：
- **增长分析（DataFinder）**：用户行为分析与增长归因
- **A/B 测试（DataTester）**：实验管理与分流服务
- **客户数据平台（VeCDP）**：用户标签管理、群体洞察与精准营销
- **增长引擎（GMP）**：营销自动化（MA）、智能营销引擎（CIE）、魔方落地页（H5）、创意助手（AIGC）四大板块

### 4. 高质量数据集
作为 AI 核心燃料供给站，提供从数据接入到消费的全生命周期服务，解决企业"高质量数据资产匮乏"痛点。

## 三、DataAgent 体系

### 3.1 知识引擎

把企业原本分散在数据库、对象存储、在线文档、图片、音视频和业务系统中的异构知识，转化为 **AI 可以稳定理解、检索、推理并输出的"企业知识底座"**。

#### 产品功能架构

| 层级 | 核心能力 |
|------|---------|
| **知识接入层** | 回答"知识从哪里来、怎么稳定进来"。覆盖 MySQL/Oracle/PostgreSQL 等结构化数据库；TOS、S3 兼容对象存储、FTP/SFTP/NAS 等文件系统；飞书、腾讯文档及本地上传文件 |
| **知识处理层** | 把"能存"变成"能用"。可视化编排、多表关联、清洗与标准化、OCR、ASR、图像识别、视频解析、智能打标、自定义算子接入 |
| **知识库与挖掘层** | 组织"企业记忆"。多类型知识库、灵活切片策略、知识单元管理、生命周期管理、召回策略、知识质量检测；向上延伸到 GraphRAG、知识图谱、本体建模与知识推理 |
| **智能问答与开放层** | 两类出口：面向人（智能问答、多轮会话、知识搜索、答案生成）；面向系统（标准化 API、知识调用接口、与 Agent 平台对接） |
| **管理与安全层** | 决定能否进入生产环境的关键。项目中心、多租户隔离、功能/资源/内容权限、资源管理、用量监控、操作日志、审计、数据加密与脱敏 |

#### 两个关键技术能力

- **Chunking 切片策略**：决定一份知识应该被拆成多大颗粒、保留多少上下文、是否按固定长度/标点/模型理解/上下文感知方式切分
- **本体建模（Ontology）**：不只是打标签，而是在更高层定义"对象—属性—关系"的业务语义框架，让系统知道"谁和谁有关系、这种关系叫什么、能不能继续推理"

#### 典型应用场景
- **通用行业**：打造统一知识入口与 RAG 精准问答
- **汽车行业**：构建全生命周期智能车书，赋能售后维修与销售
- **金融行业**：以 GraphRAG 实现政策比对、投研沉淀与风控反洗钱
- **政府行业**：提供权威政策问答与办事流程导航
- **泛互/消费**：自动构建商品知识库，实现"一客一策"千人千面推荐

### 3.2 智能分析 Agent

基于 **AI 深度思考与大数据分析** 的洞察研究专家，融合大模型深度思考能力，运用交叉分析与归因分析，以智能问数为基础，完成大纲规划并执行任务，最终形成含 **结论、多维数据分析及建议** 的深度研究报告。

#### 产品功能架构
- **多样化输出**：门户统一 Agent 入口、Oncall 群/用户飞书群、Agent 机器人等多渠道交互
- **产品能力层**：
  - **数据引擎**：知识速通（业务速通、问数据、数据资产）、数据速查（全局知识、问口径、IM/文档对话）、资产速找（图片/视频生成）、创作生成（文档/PPT 撰写）
  - **深度研究**：通过 Dynamic Planner、Multi-Agent 架构实现推理思考，支持行研分析报告、文件深度解读、多模态输出
  - **AI + 综合应用**：洞察归因、智能决策、智能解读、移动智能舱
  - **智能决策**：多轮对话（多模态分析）、反馈澄清（归因分析、个性推荐）
- **核心引擎层**：基于 EMR、ByteHouse、LAS 等数据引擎，结合 DataWind 数据资产和企业知识引擎构建核心能力

#### 典型应用场景
- **数据问答**：统一调度企业内部结构化数据、非结构化知识（本地文档、400 通话记录、车联网数据）和互联网/行业报告数据
- **深度研究**：自动生成业务分析框架，通过 SQL 与 Python 代码洞察数据。例："分析 A 门店上月销售额下降 10% 的原因"
- **AI+BI 联合洞察**：自动化异常预警与根因定位。例："总结一下这个看板我该关注什么"

### 3.3 智能营销 Agent

**核心定位**：新一代 AI 营销决策中枢，融合画像分析、实时意图识别与动态策略生成能力，驱动营销全链路升级，实现 **"一客一策"**。

#### 产品功能架构
- **底层数据**：整合企业客户数据、企业知识文档、第三方数据接入等多源数据
- **数据处理**：数据接入、清洗、结构化整合、非结构化数据接入
- **智能基座**：客户深度洞察（立体画像）+ 企业知识引擎（录入管理、挖掘检索）
- **应用能力**：
  - **公域营销洞察**：行业趋势、产品卖点、市场人群、竞品跟踪
  - **营销策略助手**：智能圈人、智能匹配（商品/内容）、营销内容生成
  - **营销互动助手**：智能导购/销售/客服场景

#### 核心功能模块

**营销策略助手**
- **智能圈人**：融合实时/标签/行为数据，例："帮我找一批高价值用户"
- **智能匹配**：自动生成"时机-渠道-内容"最优组合，例："帮我分析 SPU 营销潜力"

**营销互动助手**
- **实时客户画像**：销售对话中查看历史购买记录
- **自动生成话术**：秒级生成专家级话术，例：用户询问换手机时生成个性化推荐
- **智能商品推荐**：实时权益查询 + 推荐

**一客一策决策引擎**
- **用户挖掘**：个体意向预测、客户意图分析
- **策略生成**：服务关怀、营销建议、商品推荐、长期培育规划

#### 典型应用场景
- **汽车行业**：多源车联网数据 → 车辆健康监控、设计缺陷定位、产品迭代
- **零售行业**：某咖啡品牌联合火山引擎构建 AI Agent 体系，订单转化率 **提升 22%**

### 3.4 用户研究 Agent

**核心定位**：新一代用户精准洞察分析与决策助手，通过 AI 技术对用户行为数据、用户画像进行深度理解，提供 **AI 行为深度研究、AI 虚拟用户调研、AI 素材评估** 等核心能力。

#### 核心能力模块

**AI 行为研究**
- **日志增强理解**：JSON 埋点 → 自然语言描述 + 可视化用户动线
- **用户行为深度洞察**：自动化输出用户洞察分析报告
- **行为数据追溯**：结论关联埋点，一键验证

**AI 虚拟调研**
- **调研人群生成**：基于公域画像标签（年龄/性别/职业/学历/地域）生成虚拟数字人
- **调研问卷配置**：可视化界面 + 图片附件
- **定性/定量调研**：典型用户深度访谈 + 全量回收统计

**AI 素材评估**：针对图文、短视频等，从内容、公域用户维度预测目标接受度

**AI 虚拟人会话评估**：高仿真 AI 客户/陪练对象，员工实战演练

#### 典型应用场景
- **泛互联网**：某短剧 APP 通过该功能实现 **UV 增长 5w+**，裂变分享率达 **30%**
- **大消费/汽车**：虚拟生成 500 用户，**1 小时内**产出调研分析报告，指导电池技术、智能化体验优化

- 高效混合检索

已与 ArkClaw 实现适配联动，为 Agent 打造 **精准、低成本、安全** 的记忆底座。
